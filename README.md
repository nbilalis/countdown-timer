Ο κώδικας που πρέπει να προσθέσετε πρέπει να μπει στα σημεία με σχόλιο της μορφής `// TODO:`

Κάθε `// TODO:` σχόλιο πρέπει να αντικατασταθεί από **μία μόνο** εντολή.
